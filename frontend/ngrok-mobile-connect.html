Connect to your local development server from mobile devices using ngrok.

Note: Make sure your local development server is running with npm run dev and ngrok is running with npx ngrok http 3000.

How to Use

1 Start your Next.js server with npm run dev

3 The URL will be auto-detected or you can enter it manually
4 Scan the QR code with your mobile device
